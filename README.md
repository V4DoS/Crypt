# B4DoS Crypt Tool

Простое и надёжное приложение для шифрования и расшифровки файлов с помощью **гибридного шифрования** (RSA-4096 + AES-256-GCM).

Оригинальные файлы удаляются после успешного шифрования, а зашифрованные файлы (.B4DoS) — после успешной расшифровки.

## Возможности

- Графический интерфейс (Tkinter) — без консоли
- Две основные операции:
  - Зашифровать папку (и все подпапки рекурсивно)
  - Расшифровать папку (все файлы с расширением .B4DoS)
- Автоматическое удаление исходных файлов после успешной операции
- Защита от случайного перезаписывания (пропускает уже существующие файлы)
- Поддержка пароля на приватном ключе
- Чёткие сообщения и подтверждение перед запуском
- Подходит для защиты личных документов, фото, баз данных и т.д.

## Как это работает

1. **Шифрование**  
   - Генерируется случайный ключ AES-256 для каждого файла  
   - Файл шифруется в режиме AES-GCM  
   - Ключ AES шифруется публичным ключом RSA-4096  
   - Результат сохраняется рядом с суффиксом `.B4DoS`  
   - Оригинальный файл удаляется

2. **Расшифровка**  
   - Приватным ключом расшифровывается AES-ключ  
   - Файл расшифровывается с проверкой целостности (GCM-тег)  
   - Создаётся исходный файл  
   - Зашифрованный файл `.B4DoS` удаляется

## Требования

- Windows (тестировалось на 10/11)
- Для запуска .exe — никаких зависимостей (всё внутри)

Для разработки и сборки:

- Python 3.9–3.12
- библиотеки: `cryptography`, `pyinstaller`

## Установка и использование

1. Скачайте готовый файл `B4DoS Crypt Tool.exe` из [Releases](https://github.com/ваш-username/b4dos-crypt-tool/releases)
2. Запустите приложение
3. Выберите действие:
   - «Зашифровать папку» → укажите публичный ключ (.pem)
   - «Расшифровать папку» → укажите приватный ключ (.pem)
4. Выберите папку → подтвердите запуск

**Важно:**  
- Храните приватный ключ в надёжном месте  
- Потеря приватного ключа = невозможность расшифровать файлы навсегда  
- Рекомендуется делать резервные копии ключа на нескольких носителях
